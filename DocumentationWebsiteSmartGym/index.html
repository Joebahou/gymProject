Motivation

Our main motivation is to improve the overall experience for
The Gym Owner, The Gym Instructor and The Gym Trainee.
Our System provides the following features:
Know which machines are occupied in real time.
Ask for the instructor help.
Save the log of your trainings to ensure an effective progress.
Improve training efficiency by saving the machines in advance.
Get statistics for the machines utilization and the busiest hours.
Know which machine requires maintenance.

Statistics

The motivation for presenting statistics in the project is to streamline the acitivites of all positions at the Gym.
Below are the available statistics:

Trainee's Statistics

Our goal is to improve the trainee health. Our project helps the trainee to track his activity in the gym
Which can be used for getting a good progress and achieve his/hers personal goals.
We provide three features:
1) You can get detailed statistics for every device you used. For each use you can find out: date, number of sets, number of reps, length of use.
2) You can select a date range and see a graphic presentation of your progress in every machine you used during that time.
3) You can see your progress Compared to other trainees with the same age and gender.

  • Trainee's Statistics Homepage
  • Detailed usage per machine
  • Progress per machine

Instructor's Statistics

The instructor can get the statistics of his/her trainees.
The instructor can also get the distribution of his trainess by age and gender.

  • Instructor's Statistics Homepage
  • Go to trainee selection menu
  • Select the trainee you want to check his\hers statistics.
  • See your trainees age distribution
  • See your trainees gender distribution

Gym Owner's Statistics

We provide the below statistics for the gym owner:
1) Select a date range and find out which machines are the most popular in the gym in order to make the right decision regarding machines purchasing.
2) Check which hours are popular for better planning of the instructors shifts.

  • Gym owner's Statistics Homepage
  • Machine's popularity
  • Hour's popularity

Technology

User App

1)Listens to SignalR in order to update information on real time.
2) Connects to SQL database in order to get and set information in the sql tables.


Machine App(The device)

Sends messages to the Iot Hub.
The Iot Hub forwards the messages to the Event Hub .
The Event Hub trigger function sends relevent messages using the SignalR according to the information from the device.
