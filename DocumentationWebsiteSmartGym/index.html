Motivation

Our main motivation is to improve the overall experience for the Gym Owner, Gym Instructors and Gym Trainees.

Our System provides the following features:
1) Know which machines are occupied in real time.
2)Ask for the instructor help.
3)Save the log of your trainings to ensure an effective progress.
4)Improve training efficiency for gym instructors by saving the machines in advance for their trainees.
5)Get statistics for the machines utilization and the busiest hours.
6)Know which machine requires maintenance.

Machine App

The machine app is our device. It controls all the gym machines and allows the user to save the usage and progress on each machine.

The app allows to:
1)Scan the personal qrcode of the trainee in order to start and stop using a machine.
- When there is other trainee in the schedule in a few minutes, you get a message.
2)Save information about using a machine: when the trainee starts using the machine, he enters info about the usage.
3)Call for help from a machine.
4)Alert for a broken machine.
5)Change status of the machine(broken/not broken) - only for owner.

  • Welcome page of the machine app. Click on the button to see al machines.
  • Choose machine to start using the machine
  • Home page of a machine.Click start scan the than scan your qrcode to start and stop using the machine.
  • When you start using the machine - you need to enter info usage.
  • Option to change the status of the machine when there is an alert on broken machine.
  • Only the owner can change the status of a machine.

Statistics

The motivation for presenting statistics in the project is to streamline the acitivites of all positions at the Gym.
Below are the available statistics:

Trainee's Statistics

Our goal is to improve the trainee health. Our project helps the trainee to track his activity in the gym
Which can be used for getting a good progress and achieve his/hers personal goals.

We provide three features:
1) You can get detailed statistics for every device you used. For each use you can find out: date, number of sets, number of reps, length of use.
2) You can select a date range and see a graphic presentation of your progress in every machine you used during that time.
3) You can see your global progress compared to other trainees with the same age and gender. The yello column will represent the current trainee.

* The score in the own and global progress is calculated considering the weight/speed, sets,reps that the trainee is filling when using a machine.

  • Trainee's Statistics Homepage
  • Detailed usage per machine
  • Progress per machine
  • Comparison between the trainee progerss and other trainees with the same age and gender.

Instructor's Statistics

The instructor can get the statistics of his/her trainees.
The instructor can also get the distribution of his trainess by age and gender.

  • Instructor's Statistics Homepage
  • Go to trainee selection menu
  • Select the trainee you want to check his\hers statistics.
  • See your trainees age distribution
  • See your trainees gender distribution

Gym Owner's Statistics

We provide the below statistics for the gym owner:
1) Select a date range and find out which machines are the most popular in the gym in order to make the right decision regarding machines purchasing.
2) Check which hours are popular for better planning of the instructors shifts.

  • Gym owner's Statistics Homepage
  • Machine's popularity
  • Hour's popularity

Technology


User App

1)Listens to SignalR in order to update information on real time.
2) Uses http trigger function app in order to get and update the data in the SQL database.


Machine App(The device)

1) Sends messages to the Iot Hub ->.
->The Iot Hub forwards the messages to the Event Hub -> .
-> The Event Hub trigger function sends relevent messages using the SignalR according to the information from the device.

2)Listens to SignalR to get messages on relevant info. 3)Uses http trigger function app in order to get information from the SQL database.
